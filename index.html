How this prototype is built

Answers are not generated live by an AI model. Each is a pre-written, human-vetted entry tied to a published source — a deliberate control, since an incorrect Medicare answer carries real cost to a member. Incoming questions are matched to the knowledge base by keyword; when no answer clears the confidence threshold, the assistant escalates to a licensed human rather than guessing.

This mirrors how a process owner would scope such a tool: a defined intake, an explicit knowledge base, a control for low-confidence cases, and measurable outcomes.
